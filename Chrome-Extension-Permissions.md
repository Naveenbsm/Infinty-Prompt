## Chrome Extension Permissions
This document explains the permissions requested by Infinity Prompt Optimization and their intended usage

## ActiveTab
Used to interact with the currently active webpage when the user initiates an action

## Scripting
Allows dynamic injection of content scripts and UI components in response to user ineraction

## Side Panel
Enables the use of Chrome side panel for extended prompt generation features

## Storage
Used to save user preferences such as disabled domains and UI settings

## Host Permissions
Required to allow the extension to function consistently across any website

No data is accessed without explicit user interaction

## Permission Philosophy
- All permissions are strictly necessary for core functionality
- No unused or hidden permissions are requested
- Permissions are never used for tracking, analytics or advertising

Infinity Prompt Optimization follows chrome principle of least privilege wherever possible
