## Privacy and Security
User privacy and data security are core principles of Infinity Prompt

## Data Handling Policy
- The extension only processes text explicitly selected or entered by the user
- No background data scraping or passive data collection occurs
- No user data is stored, logged or shared

## API Communication
- All AI processing is performed via secure HTTPS requests
- API keys are never exposed to content scripts or UI layer
- Requests are sent only when triggered by user action

## Local Storage Usage
- Chrome local storage is used solely for user preferences
- No sensitive or personal information is stored
- Domain based exclusions arte stored locally on the users device

## Permissions Justification
- Permissions are minimized and purpose driven
- No permissions are used for tracking or analytics
- The extension does not monitor browsing behavior

## Security Philosophy
Infinity Prompt Optimization is designed to Be

- Transparent
- Non invasive
- Secure by default

User trust is treated as a critical system dependency
